# Write Insight Statements from Raw Findings

**Role:** Both

## Prompt

```
Here are my raw research findings:

[Paste findings]

Turn these into insight statements using this structure:
"[User type] [behavior or belief] because [underlying motivation or context], which means [implication for design]."

For each insight:
- Rate the evidence strength (strong / moderate / emerging)
- Note what would make the evidence stronger
- Suggest one design implication

Give me 5-8 insight statements, prioritized by strength of evidence.
```
