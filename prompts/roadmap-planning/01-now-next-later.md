# Build a Now / Next / Later Roadmap

**Role:** Facilitator

## Prompt

```
I need to organize our design work into a Now / Next / Later roadmap.

Here's everything on our plate:
[List all initiatives, features, debt, and research]

Our team: [size and composition]
Our strategy: [what are we trying to achieve this quarter/half?]
Key constraints: [dependencies, deadlines, unknowns]

Help me:
1. Sort items into Now (this sprint/month), Next (next 1-3 months), Later (3-6 months)
2. For each bucket, explain the rationale (why now? why wait?)
3. Identify dependencies between items
4. Flag anything that's on the list but probably shouldn't be
5. Write a one-paragraph narrative that turns this list into a story ("First we'll... which enables... so that eventually...")
```
