# Generate "How Might We" Notes

**Role:** Participant

## Prompt

```
Our sprint challenge is: "[paste the HMW question]"

I need to generate How Might We notes during our sprint. Help me think divergently by generating 15 HMW questions across these angles:
- Reframe the problem (what if the opposite were true?)
- Amplify what's already working
- Remove a constraint entirely
- Serve an extreme user
- Break the problem into smaller parts

Make each one specific enough to spark a real idea, not so broad it could mean anything.
```
