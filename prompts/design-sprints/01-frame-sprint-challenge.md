# Frame the Sprint Challenge

**Role:** Facilitator

## Prompt

```
We're running a design sprint on [topic/problem]. Here's what we know:

- Business context: [brief description]
- Users affected: [who]
- Current pain points: [what's broken or missing]
- Constraints: [time, tech, budget, politics]

Help me write a sprint challenge question in the format: "How might we [outcome] for [user] so that [impact]?"

Give me 3 versions: one ambitious, one pragmatic, one that reframes the problem entirely. For each, explain what it would focus the team on and what it would leave out.
```
