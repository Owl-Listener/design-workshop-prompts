# Write a Test Script for Prototype Friday

**Role:** Facilitator

## Prompt

```
We've built a prototype for [brief description]. We're testing with [number] users on Friday.

Our top 3 questions to answer:
1. [question]
2. [question]
3. [question]

Write a 30-minute usability test script with:
- An intro that puts participants at ease and sets expectations
- 4-5 task scenarios that test our questions (phrased as realistic goals, not instructions)
- Probing follow-up questions for each task
- A debrief section to capture reactions
- A note-taking template I can hand to observers
```
