# Sketch Session: Generate Solution Concepts

**Role:** Participant

## Prompt

```
I'm in a design sprint sketch session. Our challenge is: "[HMW question]"

Help me brainstorm 6 distinct solution directions. For each one, give me:
- A descriptive name (2-3 words)
- The core idea in one sentence
- What makes this approach different from the others
- The biggest assumption it relies on

I'm looking for range, not refinement. Push me past the obvious solutions.
```
