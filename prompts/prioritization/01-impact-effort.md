# Facilitate an Impact-Effort Prioritization

**Role:** Facilitator

## Prompt

```
We need to prioritize these design opportunities:

[List your opportunities/features/ideas]

For each one, help me assess:
- User impact (how much does this improve someone's experience? low / medium / high)
- Business impact (does this move a metric leadership cares about? low / medium / high)
- Effort (design + engineering + testing, roughly: small / medium / large)
- Risk (what could go wrong and how bad would it be?)

Then place them in a 2x2 matrix (high impact + low effort = do first) and explain your reasoning. Flag any where the "obvious" quadrant might be misleading.
```
