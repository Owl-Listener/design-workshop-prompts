# Run a "MoSCoW" Prioritization Workshop

**Role:** Facilitator

## Prompt

```
I'm facilitating a MoSCoW prioritization session with [team/stakeholders]. We're prioritizing [what] for [timeline].

Here are all items to prioritize:
[List them]

Help me:
1. Write a 2-minute explanation of MoSCoW I can read to the group (Must have / Should have / Could have / Won't have this time)
2. Pre-sort these items with a rationale for each, so I have a starting position to react to
3. Identify the items that will cause the most debate and prepare a question for each that helps resolve it
4. Set a rule for the maximum percentage of effort that can be "Must have" (typically 60%)
5. Write a summary template I can fill in after the session
```
