# Evaluate Trade-Offs Honestly

**Role:** Both

## Prompt

```
We're considering this trade-off:

[Describe the decision: e.g., "Ship a simpler version now vs. build the full feature over 3 months"]

Help me think through this clearly:
1. What do we gain with each option? (Be specific, not just "speed" or "quality")
2. What do we lose with each option? (Be honest about the real costs)
3. Who is affected most by each choice and how would they feel about it?
4. What's reversible and what isn't?
5. In 6 months, which decision will we be glad we made?

Don't tell me what to choose. Help me see the full picture.
```
