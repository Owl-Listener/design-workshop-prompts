# Resolve a Prioritization Deadlock

**Role:** Facilitator

## Prompt

```
Our team is deadlocked on prioritization. The debate is between:

Option A: [describe]
Option B: [describe]

The people arguing for A care about: [their reasoning]
The people arguing for B care about: [their reasoning]

Help me facilitate a resolution:
1. Name the underlying values conflict (what does each side actually value?)
2. Is there a "both/and" option that addresses both concerns?
3. If we must choose: what's a fair decision framework we can agree on before applying it?
4. A time-boxed experiment we could run instead of deciding permanently
5. The single question that, if answered, would make the choice obvious
```
