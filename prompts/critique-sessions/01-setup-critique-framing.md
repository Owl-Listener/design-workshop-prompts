# Set Up a Critique with Clear Framing

**Role:** Facilitator

## Prompt

```
I'm presenting [design/concept] at a critique session. The audience is [who]. The work is at [stage: early exploration / mid-fidelity / near-final].

Write me a critique setup brief that includes:
- A 2-sentence framing of what this design is trying to do
- 3 specific questions I want feedback on (calibrated to the stage of the work)
- What kind of feedback is helpful right now and what isn't
- A suggested format for the session (silent review + discussion, round-robin, etc.)
- A time structure for a [30/45/60]-minute session
```
