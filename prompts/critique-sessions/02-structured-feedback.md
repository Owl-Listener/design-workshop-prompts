# Generate Structured Feedback on a Design

**Role:** Participant

## Prompt

```
I need to give critique feedback on this design: [describe or paste screenshot description]

The designer asked for feedback on: [their specific questions]

Help me structure my feedback using this framework:
1. What's working well and why (be specific, not just "looks good")
2. What's confusing or creating friction, and for whom
3. Questions I'd want to answer before going further
4. One alternative approach worth exploring, with a clear rationale

Keep the tone constructive. I want to push the work forward, not just react to it.
```
