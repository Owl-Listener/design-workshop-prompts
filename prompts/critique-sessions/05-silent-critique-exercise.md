# Facilitate a "Silent Critique" Exercise

**Role:** Facilitator

## Prompt

```
I want to run a silent critique where reviewers write feedback independently before discussing. We have [number] reviewers and [number] designs to review.

Design the exercise for me:
1. A brief explanation I can read to the group
2. The exact prompts/questions each reviewer should answer in writing (keep to 3-4)
3. How long to give for silent review vs. discussion
4. How to facilitate the discussion so quieter voices get heard
5. A template I can paste into a shared doc for reviewers to fill in
```
